DesertEmail

A from-scratch pure-Rust email server for tiny machines — zero dependencies, one binary.

Install (one command)

curl -fsSL https://YOUR-SITE/install.sh | sh

Downloads a prebuilt static binary — no local compile, works on low-memory systems (including ~128 MB RAM).

Features

  • From-scratch pure Rust — zero external crates
  • ~488 KB static binary after strip/LTO
  • SMTP + IMAP + webmail + DKIM + outbound MTA
  • Runs on 128 MB RAM devices (e.g. Raspberry Pi Zero)
  • Prebuilt musl Linux + macOS binaries via GitHub Releases

What the installer does

  • Detects OS/arch and fetches the matching release binary
  • Verifies SHA-256 against SHA256SUMS
  • Installs under ~/.desertemail (override with DESERTEMAIL_PREFIX)
  • Interactive wizard for domain, admin user, ports, webmail, DKIM
  • Writes config.toml; optionally installs a systemd unit
  • Headless-safe: curl | sh works with env-var defaults

Source & releases: github.com/bitfent/desertemail

Binaries are published on GitHub Releases; this site only serves the landing page and install.sh.
